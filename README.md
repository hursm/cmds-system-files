## Overview

CMDS (커맨드스페이스) is a comprehensive Personal Knowledge Management (PKM) system built on Obsidian. This repository contains the 5 core system files that provide complete guidance for AI agents and human users.

### CMDS Process

```
Connect → Merge → Develop → Share
```

- **Connect**: Discover ideas by connecting interests and topics (100)
- **Merge**: Build theoretical foundations by integrating knowledge (200)
- **Develop**: Collect data and apply methodologies (300-600)
- **Share**: Disseminate creative work and outputs (700-800)
